# Mattermost

```bat
rem チームの削除
mmctl team delete sample_team

rem チームのアーカイブ
mmctl team archive sample_team

rem チームのリストア（アーカイブからの復元）
mmctl team restore sample_team

rem チームへのユーザー追加
mmctl team users remove sample_team t_sample@trancom.co.jp username

rem チームからユーザー削除
mattermost team remove sample_team t_sample@trancom.co.jp

rem チャンネルの削除
mmctl channel delete sample_team:sample_channel

rem チャンネルのアーカイブ
mmctl channel archive sample_team:sample_channel

rem チャンネルのリストア（アーカイブからの復元）
mmctl channel unarchive mynewteam:mynewchannel

rem チャンネルへのユーザー追加
mmctl channel users add sample_team:sample_channel t_sample@trancom.co.jp

rem チャンネルからユーザー削除
mmctl channel users remove sample_team:sample_channel t_sample@trancom.co.jp

rem ユーザーの追加
mmctl user create --email t_sample@trancom.co.jp --username t_sample --password t.sample --firstname 太郎 --lastname サンプル

rem ユーザーの削除
mmctl user delete t_sample@trancom.co.jp

rem ユーザーの無効化
mmctl user deactivate t_sample@trancom.co.jp

rem ユーザーの有効化
mmctl user activate t_sample@trancom.co.jp
```
