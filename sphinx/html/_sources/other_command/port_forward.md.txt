# ポートフォワード

```bat
rem ポートフォワード　表示
netsh interface portproxy show all

rem ポートフォワード　追加
set wsl2_ip=172.23.61.180
netsh interface portproxy add v4tov4 listenport=1433 listenaddress=0.0.0.0 connectport=1433 connectaddress=%wsl2_ip%

rem ポートフォワード　削除
netsh interface portproxy delete v4tov4 listenport=1433 listenaddress=0.0.0.0
```
