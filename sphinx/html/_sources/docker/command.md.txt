# コマンド一覧

## Docker

```bash
# プロセス一覧
docker ps -a

# コンテナ内部へ入る
docker exec -it $container_name bash

# ホストからコンテナへコピー
docker cp $file_name $container_name:$path

# コンテナからホストへコピー
docker cp $container_name:$path $file_name

# vimをインストール
apt-get update && apt-get install vim

# イメージのバックアップ
docker save $image_name > $image_name.tar

# イメージの復元
docker load < $image_name.tar

# イメージ一覧
docker image ls

# イメージの削除
docker image rm {image_name}:{tag}

# コンテナの削除
docker rm {image_name}:{tag}

# ボリューム一覧
docker volume ls

# ボリューム削除
docker volume rm {volume_name}

# ビルドキャッシュの削除
docker system df
docker builder prune
```
