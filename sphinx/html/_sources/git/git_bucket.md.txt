# Git Bucket

## 利用方法

1. ブランチ間の差分確認

    ```text
    http://hostName:portNo/[リポジトリ名]/compare/[比較対象1]...[比較対象2]
    ```

    例 > <https://compass-upd.trancom.co.jp/gitbucket/m_morioka/sample_python/compare/master...m_morioka>

## forkしたリポジトリをprivateにする方法

1. forkしたリポジトリをbareでクローンする

    ```bash
    git clone --bare https://compass-upd.trancom.co.jp/gitbucket/git/y_kasahara/my_sphinx.git
    ```

1. gitbucketでprivateリポジトリを作成する（例：my_sphinx_private）

1. priavteリポジトリにプッシュする

    ```bash
    git push --mirror https://user_name:password@compass-upd.trancom.co.jp/gitbucket/git/y_kasahara/my_sphinx_private.git
    ```

1. gitbucketでリポジトリ名を戻す（リポジトリ名：my_sphinx）
