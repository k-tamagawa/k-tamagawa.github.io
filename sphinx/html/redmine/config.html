5.1. 設定確認¶

5.1.1. デバッグログの出力方法¶

  • production.rbの編集
    保存場所：C:\redmine\config\environments\production.rb

    #キャッシュはしないようにする(trueからfalseに変更)
config.cache_classes = false

#ログ設定を追加
config.log_level = :debug

  • 任意のソースに以下を追加する。

    logger.debug("ログアウト - #{home_url} ")

5.1.2. ルーティングの確認¶

  • 以下のファイルを確認
    C:\redmine\config\routes.rb
