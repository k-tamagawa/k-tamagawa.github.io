# Windows 10 拡張機能インストール

## [アプリと機能]の[機能と追加]で機能追加する方法

1. `gpedit.msc`を実行する。

1. [**コンピューターの構成 > 管理用テンプレート > Windowsコンポーネント > Windows Update**]の[**インターネット上のWindows Updateに接続しない**]を`無効`にする。

1. [**コンピューターの構成 > 管理用テンプレート > システム**]の[**オプションコンポーネントのインストールおよびコンポーネントの修復のための設定を指定する**]を`有効`にする。

1. 上記の画面左下のオプションで、[**Windows Server Update Services (WSUS) の代わりに、Windows Updateから修復コンテンツとオプションの機能を直接ダウンロードする**]に`チェック`する。

1. `regedit`を実行する。

1. [**HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate**]の[**DoNotConnectToWindowsUpdateInternetLocations**]を`1`から`0`に変更する

## Microsoft Storeアプリのインストール

Microsoft Storeアプリはポリシーでインストールできない。  
msixbundleファイルでStoreアプリをインストールすることはできるが、依存関係があるライブラリが必要となった場合、  
Microsoft Storeからダウンロードされるためエラーとなる。  
これで回避できるかわからないが、この設定をした状態でインストールできたことがあったため記載しておく。

1. [**アプリと機能の機能と追加で機能追加する方法**]に記載の設定をする。

1. [**コンピューターの構成 > 管理用テンプレート > Windows コンポーネント > ストア**]に移動する。

1. [**Microsoft Store のすべてのアプリを無効にする**]を`無効`にする。

1. [**Microsoft Store アプリケーションをオフにする**]を`無効`にする。
