12.1. Windows 10 拡張機能インストール¶

12.1.1. [アプリと機能]の[機能と追加]で機能追加する方法¶

  1. gpedit.mscを実行する。

  2. [コンピューターの構成 > 管理用テンプレート > Windowsコンポーネント > Windows Update]の[インターネット上のWindows Updateに接続しない]を無効にする。

  3. [コンピューターの構成 > 管理用テンプレート > システム]の[オプションコンポーネントのインストールおよびコンポーネントの修復のための設定を指定する]を有効にする。

  4. 上記の画面左下のオプションで、[Windows Server Update Services (WSUS) の代わりに、Windows Updateから修復コンテンツとオプションの機能を直接ダウンロードする]にチェックする。

  5. regeditを実行する。

  6. [HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate]の[DoNotConnectToWindowsUpdateInternetLocations]を1から0に変更する

12.1.2. Microsoft Storeアプリのインストール¶

Microsoft Storeアプリはポリシーでインストールできない。
msixbundleファイルでStoreアプリをインストールすることはできるが、依存関係があるライブラリが必要となった場合、
Microsoft Storeからダウンロードされるためエラーとなる。
これで回避できるかわからないが、この設定をした状態でインストールできたことがあったため記載しておく。

  1. [アプリと機能の機能と追加で機能追加する方法]に記載の設定をする。

  2. [コンピューターの構成 > 管理用テンプレート > Windows コンポーネント > ストア]に移動する。

  3. [Microsoft Store のすべてのアプリを無効にする]を無効にする。

  4. [Microsoft Store アプリケーションをオフにする]を無効にする。
